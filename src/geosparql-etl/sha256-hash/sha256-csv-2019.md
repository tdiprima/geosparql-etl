1. Generate the script `python sha256-csv-2019.py`

2. This creates: `compute_sha256_hashes.sh`

3. Copy that script to the machine with the SVS files and run it: `./compute_sha256_hashes.sh`

4. This will create: `slide_hashes.json`

5. Then you'll bring `slide_hashes.json` back to use in Part 2 to update the .ttl.gz files!

6. Finally: `python update_ttl_gz_from_json.py`
